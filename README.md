# 무한 개소리 생성기
Gemma3 Pretrained 모델 기반의 무한 개소리 생성기

# 작동 루프
1. 최초 시드 입력 (또는 시드 랜덤 생성)
2. 시드를 모델에 입력 후 응답 생성
3. 응답 완료 결과물에서 몇개의 단어를 랜덤으로 골라 다음 시드로 선중
4. temperature 값 랜덤 변경
5. 2번 단계부터 다시 반복

# 용도
끊임없이 그럴듯하지만 아무 의미가 없는 문장이 흐르는것을 보며, 전기를 불태우는 디지털 캠프파이어 불멍을 즐길 수 있습니다. 
장시간 감상 시 부작용으로 지능이 하락할 가능성이 있습니다.

# 사양
RTX 5070 laptop (VRAM 8GB) 환경에서 원할한 구동이 가능했습니다.
